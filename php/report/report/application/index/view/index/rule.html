举报须知

一、 受理范围

本网站受理涉嫌违反《全国人民代表大会常务委员会关于维护互联网安全的决定》、《中华人民共和国刑法》、《中华人民共和国治安管理处罚法》、《互联网信息服务管理办法》等法律法规有关条款规定，利用互联网或针对网络信息系统从事违法犯罪行为的线索，具体行为包括：

1、利用互联网捏造或者歪曲事实、散布谣言，扰乱社会秩序的；

2、利用互联网建立淫秽色情网站、网页，提供淫秽站点链接，传播淫秽色情信息，组织网上淫秽色情的；

3、利用互联网进行诈骗的；

4、干部违纪方面的举报。

　

二、注意事项

1、请您根据举报的具体内容，在各举报类别的范围内进行举报。

2、请您自觉遵守中华人民共和国宪法和法律。

3、您应对举报内容的真实性、客观性负责，不得捏造事实、制造假证、诬告陷害他人。借检举诬告陷害他人，会受到严肃处理，甚至会被追究法律责任。

4、举报应当实事求是，如实提供被举报人的有关情况和犯罪事实。

5、本网站鼓励实名举报。为便于提供有针对性的指导帮助，请详细填写联系方式。举报人填写的所有举报信息将依法受到保护。

三、为了您的申请能及时被受理，请特别注意以下事项：

1、申请者请完整、详细填写相关举报表单，不得遗漏每一项，否则不予处理，完成举报后请耐心等候按管理人员受理；

2、资料不齐的，理由不充分的申请，反复提交将不予处理；

3、出于媒体舆论监督属性，并非所有申请都必须受理，请谅解；

4、证件必须齐全，上传的电子版证件、材料必须清晰。举报内文中请附上相关标题和信息链接等，并请详细解释原因，必要的需列上证据。一旦申请不予撤销，请慎重申请！

5、处理时间一般为2-3个工作日。最终解释权归举报投诉系统所有。
